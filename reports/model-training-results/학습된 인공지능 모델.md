## 모델 설계도 (Model Architecture)

이 프로젝트는 다음 4가지 모델을 포함하고 있습니다.

### 모델 상세 구조
| 모델명 | 주요 특징 | 사용 용도 |
| :--- | :--- | :--- |
| **LSTM** | 순차적 데이터 처리에 최적화된 RNN 기반 모델 | 시퀀스 데이터 분석 |
| **Text-CNN** | n-gram 특징 추출을 위한 합성곱 신경망 | 텍스트 분류 |
| **Transformer** | Attention 메커니즘을 이용한 병렬 연산 모델 | 자연어 이해 및 생성 |
| **KLUE-BERT** | 한국어 사전 학습된 Transformer 모델 | 한국어 감성분석/개체명 인식 |

### 모델별 설계도

**1. LSTM**
| 항목 | 값 |
| :--- | :--- |
| **Backbone Model** | LSTM |
| **Embedding Dimension** | 128 |
| **Hidden Size** | 128 (per layer) |
| **Num Layers** | 2 |
| **Bidirectional** | False |
| **Dropout** | 0.3 |
| **Output Node** | 2 |
| **Class** | retain, churn_signal |
| **Loss Function** | Focal Loss |
| **Optimizer** | AdamW |

**2. Text-CNN**
| 항목 | 값 |
| :--- | :--- |
| **Backbone Model** | TextCNN |
| **Embedding Dimension** | 256 |
| **Num Filters** | 256 (per kernel) |
| **Kernel Sizes** | 2, 3, 4 |
| **Dropout** | 0.3 |
| **Output Node** | 2 |
| **Class** | retain, churn_signal |
| **Loss Function** | Focal Loss |
| **Optimizer** | AdamW |

**3. Transformer (Scratch)**
| 항목 | 값 |
| :--- | :--- |
| **Backbone Model** | Transformer Encoder (Scratch) |
| **Embedding Dimension** | 128 |
| **Multi-Head Attention** | 4 Heads |
| **Encoder Layers** | 2 Layers |
| **FeedForward Dimension** | 256 |
| **Dropout** | 0.1 |
| **Output Node** | 2 |
| **Class** | retain, churn_signal |
| **Loss Function** | Focal Loss |
| **Optimizer** | AdamW |

**4. KLUE-BERT**
| 항목 | 값 |
| :--- | :--- |
| **Backbone Model** | KLUE-BERT |
| **Transformer Layer 수** | 12 |
| **Hidden Size** | 768 |
| **Attention Head 수** | 12 |
| **Dropout** | 0.1 |
| **Output Node** | 2 |
| **Class** | retain, churn_signal |
| **Loss Function** | Focal Loss |
| **Optimizer** | AdamW |
